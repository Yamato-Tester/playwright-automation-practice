# Playwright Automation Testing Project

Dự án thực hành Automation Test tích hợp UI và API sử dụng Playwright & JavaScript.

## 🚀 Tính năng nổi bật
- **UI Testing**: Xử lý dữ liệu động, Sorting, và luồng Checkout trên SauceDemo.
- **API Testing**: Thực hiện đầy đủ các phương thức GET, POST, PUT, DELETE.
- **Hybrid Testing**: Kết hợp API để tạo dữ liệu và kiểm tra trên giao diện Web.
- **Reporting**: Tự động chụp ảnh (Screenshot) và quay Video khi test lỗi.

## 🛠 Cài đặt
1. Clone dự án: `https://github.com/Yamato-FE-Dev/playwright-automation-practice`
2. Cài đặt thư viện: `npm install`
3. Chạy test: `npx playwright test tests/Tên file`
